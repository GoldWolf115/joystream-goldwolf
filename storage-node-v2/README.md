# Usage
<!-- usage -->
```sh-session
$ npm install -g storage-node-v2
$ storage-node COMMAND
running command...
$ storage-node (-v|--version|version)
storage-node-v2/0.1.0 darwin-x64 node-v14.8.0
$ storage-node --help [COMMAND]
USAGE
  $ storage-node COMMAND
...
```
